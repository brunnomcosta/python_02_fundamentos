# Python - Fundamentos

Repositorio com exercicios basicos de Python, organizados por arquivos numerados.

O foco deste projeto e praticar os conceitos iniciais da linguagem, com exemplos simples e didaticos.

## Conteudo

- `01-primeiro.py`: primeiros comandos e saida no terminal
- `02-dados.py`: tipos de dados basicos
- `02-dados-exercicio.py`: exercicio com tipos de dados
- `03-input.py`: entrada de dados com `input()`
- `04-concatena.py`: concatenacao e diferentes formas de exibir textos
- `05-operadores.py`: operadores aritmeticos
- `06-strings.py`: comparacao e busca em strings
- `07-slice.py`: fatiamento de strings com slice
- `08-metodos_strings.py`: principais metodos de strings
- `09-lista.py`: conceitos basicos de listas
- `10-metodo_lista.py`: metodos comuns de listas
- `ex1.py`: exercicios extras com strings
- `teste.ipynb`: anotacoes ou testes em notebook

## Como executar

No terminal, dentro da pasta do projeto:

```powershell
python 01-primeiro.py
```

Troque o nome do arquivo para executar qualquer outro exercicio.

## Requisitos

- Python 3 instalado
- Terminal ou editor com suporte a execucao de scripts Python

## Objetivo

Este repositorio serve como base de estudo para:

- entender a sintaxe basica do Python
- praticar entrada e saida de dados
- trabalhar com strings, listas e operadores
- reforcar conceitos com pequenos exercicios
